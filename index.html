Sistema CRUD com jQuery + Firebase

Aplicação web para cadastro e controle de clientes e vendedores, com persistência em tempo real no Firebase.

Como o sistema funciona

A interface apresenta a home com navegação para Integrantes, Cliente e Vendedor. Em cada tela, é possível salvar, editar, listar e excluir registros, com atualização em tempo real.

Banco de dados

A conexão com o Firebase está centralizada em js/firebase.js. Os dados são armazenados na Realtime Database e cada registro recebe um id automático gerado pelo Firebase.

Desenvolvimento
  • HTML5 e Bootstrap para interface responsiva.
  • jQuery para eventos, atualização de tabelas e CRUD.
  • Firebase Realtime Database para persistência em nuvem.
Telas principais
  • Home: apresentação do sistema e navegação.
  • Cliente: cadastro com nome, email e telefone.
  • Vendedor: cadastro com nome, salário e cargo.
Como executar
  1. Abra o projeto em um servidor local, como o Live Server.
  2. Acesse index.html.
  3. Navegue pelas páginas e teste as operações de cadastro, edição, listagem e exclusão.
